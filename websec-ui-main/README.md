# websec-ui

Фронтенд апликација за websec.

## Покретање

Исто као на часу — служити статичке фајлове са nginx или еквивалентног.

## Измене (грана `vuln/idor`)

### Брут-форс feedback у UI

`js/login.js` обрађује HTTP 429 одговор са API-ја:
- Дугме за пријаву се онемогућава током блокаде.
- Приказује се одбројавање у реалном времену (нпр. "Try again in 14s").
- Дугме се поново омогућава чим истекне блокада.
